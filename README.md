# TP Django CRUD Application

## Description

Ce projet est une application CRUD développée avec Django et Django REST Framework.
L’objectif du TP est de créer :

* Une API REST permettant la gestion des tâches
* Une interface graphique pour manipuler les données

L’application permet :

* d’ajouter des tâches
* d’afficher les tâches
* de modifier des tâches
* de supprimer des tâches

---

## Technologies utilisées

* Python
* Django
* Django REST Framework
* SQLite
* Bootstrap
* Postman

---

## Installation

```bash
git clone https://github.com/MarzoukSalma/TP_Django.git
cd TP_Django
```

Créer un environnement virtuel :

```bash
python -m venv venv
```

Activer le venv :

### Windows

```bash
.\venv\Scripts\Activate.ps1
```

Installer les dépendances :

```bash
pip install -r requirements.txt
```

---

## Lancer le projet

```bash
python manage.py runserver
```
